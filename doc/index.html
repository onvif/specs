Document preview

The links below allow to preview the documents. The document preview is using a simplified html stylesheet with similar rendering than the published pdf files.
Note that the target does not change on branching or forking. As alternative you may serve the files using a local web server. Rendering files does not activate the necessary stylesheets and hence do not show the resulting layout. Similarly for wsdl files github pages does use a Content-Type that tends to trigger a download. For the default branch of onvif/specs ONVIF provides a proxy that fixes this behavior.
